# ❇️ Claude Code（克劳德）

## 安装Claude Code

以下拿 `MacOS` 系统安装举例：

### 安装git

### 安装node

### 安装 CLI（Intel Mac 适用）

```shell
# 官方安装（推荐）
curl -fsSL https://claude.ai/install.sh | bash

# 或用 Homebrew
brew install --cask claude-code
```

> 官方安装，如遇到报错，则建议使用 `brew` 安装，一步到位，傻瓜式安装

### 绕过校验

![打开Claude报错](/src/ai/claude-cli-error.jpg)

- 修改 `.claude.json` 配置

打开 `~/.claude.json` 文件

```shell
# vscode 打开
code ~/.claude.json

# nano 打开
nano ~/.claude.json
```

增加一行代码:

```json
{
  "hasCompletedOnboarding": true
}
```

再次打开，即可正常进入：

![正常打开Claude](/src/ai/claude-cli.jpg)
