# Redis笔记

## Redis概述

高性能键值对数据库，支持的键值数据类型

- 字符串（String）
- 列表（List）
- 集合（Set）
- 有序集合（Sorted Set）
- 哈希/散列（Hash）

## Redis的应用场景

- 缓存
- 任务队列
- 应用排行榜
- 网站访问统计
- 数据过期处理
- 分布式集群架构中的session分离

## Redis的安装

- 下载安装包
- 解压安装包
- 进入解压后的目录，执行make命令
- 执行make install命令
- 进入redis目录，执行src/redis-server命令启动redis服务
- 执行src/redis-cli命令启动redis客户端

## Redis的配置文件

- daemonize yes：以守护进程方式运行
