# 文件&目录

## 解压

### .tar.bz2

```shell
tar -xjvf foo.tar.bz2
```

### .tar.gz

```shell
tar -zxvf foo.tar.gz
```

## less 浏览文件

```shell
# [pagedown]：向下翻动一页 
# [pageup]：向上翻动一页
less foo.txt
```

## chmod 设置文件权限

chmod -- 设置文件 拥有者(u)、群组(g)、其他(0) 的权限, a表示三者皆是

```shell
# r 可读 4
# w 可写 2
# x 可执行 1
# - 无权限 0

# -R : 对目前目录下的所有文件与子目录进行相同的权限变更
chmod -R 705 log.txt
# 7 = rwx = 4+2+1
# 0 = ---  
# 5 = r_x = 4+0+1
```

## find 查找/搜索文件（夹）

- [Linux的五个查找命令(find, locate, whereis, which, type)](http://www.ruanyifeng.com/blog/2009/10/5_ways_to_search_for_files_using_the_terminal.html)

```shell
# -name 符合name的文件 -iname 会忽略大小写
# ./可以不写, 默认当前目录
find ./ -name "log.txt"
find ./ -name "*og*"
```

## getfacl 查看权限

```shell
getfacl foo.txt # 也可以是目录

# 返回：
# file: foo.txt
# owner: www
# group: www
user::rw-
group::rw-
other::r--
```
