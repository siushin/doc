# 加密

- 更安全，PHP 增加对下一代密码散列算法 Argon2 的支持

```php
$password = "Hello World";
$hash = password_hash($password, PASSWORD_BCRYPT);
// $2y$10$8vNnGLQtaFQfOzqTHABsLedhkrjZWpzKUFoqkdl3TaLdGQp0JJqcy 每次刷新会改变

if (password_verify($password, $hash)) {
    echo "success";
} else {
    echo "error";
}
```
