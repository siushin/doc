# 日志

## 日志级别

ThinkPHP对系统的日志按照级别来分类记录（也称为日志类型），按照`PSR-3`日志规范，日志的级别从低到高依次为： `debug`, `info`, `notice`, `warning`, `error`, `critical`, `alert`, `emergency`。

::: tip sql日志级别
ThinkPHP额外增加了一个`sql`日志级别仅用于记录`SQL`日志（并且仅当开启数据库调试模式有效）。
:::

::: danger 系统异常日志级别
系统发生异常后记录的日志级别是`error`
:::
